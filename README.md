# Customer-Purchase-Prediction
Machine learning project to predict customer purchasing behavior using Logistic Regression.
## Project Overview
This project uses Machine Learning to predict whether a customer will purchase a product based on their age and estimated salary.
## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Matplotlib
## Machine Learning Model
Logistic Regression
## Dataset
Social Network Ads dataset.
## Features
- Data preprocessing
- Train-test split
- Model training
- Prediction
- Accuracy evaluation
- Visualization
## Output
The model predicts whether a customer will purchase a product or not.
